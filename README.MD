# Supplier Risk Assessor

## Description
This project evaluates supplier reliability and financial stability using a Python Tkinter GUI.

## Features
- Add supplier details
- Calculate supplier risk score
- Rank suppliers by risk
- Scrollable table
- Low, Medium, and High risk classification

## Technologies
- Python
- Tkinter

## How to Run
Run:
```
python supplier_risk.py
```
